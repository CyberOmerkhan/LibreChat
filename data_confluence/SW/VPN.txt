About VPN
For security and privacy reasons, some computing resources are only available when on Swarthmore's campus network and not from any computer on the Internet.  Running Virtual Private Network (VPN) software allows users to access these resources when off-campus.
Installation Instructions
For more information about VPN and detailed instructions, visit
