Padlet is a website that makes it easy to create an online bulletin board for instructors and students to share text and pictures in a variety of formats. More information:
Padlet
Instructors can create a Padlet and allow their students to access the Padlet via Moodle.
Go to your Moodle course
Turn editing on
Click
Add and activity or resource
Select
Padlet
Click
Select content
If you have an existing Padet, select the Padlet you want to use. If not, click the link to make a new Padlet and come back to select it later.
There is an option to “Create a copy for each user” if you want to make a template for each student to have their own Padlet.
Click
Save and return to course
The result is a link in Moodle to the Padlet.
