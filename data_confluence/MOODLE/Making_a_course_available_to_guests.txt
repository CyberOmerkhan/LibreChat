Title: Making a course available to guests

Version: 5

Opening up a course to guests means anyone on the Internet can access the course without needing an account.  It is possible to set an optional password for guests. If there is content under copyright or sensitive information in the course, but guest sharing is turned on, make sure to set a password to prevent open access to the material.
none
Enable guest access
Go to the
Participants Tab
(under the course title)
In the pulldown menu labeled
Enrolled Users
→ select
Enrollment Methods
If Guest access is not already listed, select
Add Method →
Guest
Access.
If Guest Access is listed but greyed out, enable it by clicking on the
eye
icon.
If you wish to set a password, click the
gear
icon and type in a password in the text box. Most people who turn on guest access set a password because the course may have fair use copyright materials or because they don't want anyone on the Internet seeing the contents of their course.
You can now copy the URL of your course main page and share it with non-Swarthmore users.  It is also possible for non-Swarthmore users to browse through the course catalog in Moodle and find your course.
When you make a course available to guests, anyone on the Internet will be able to access your course. If this is an issue, see "Adding a guest password" below to limit access to the course.
Add a guest password
To add a password to guest access, go to the
Participants Tab
(under the course title)
In the pulldown menu labeled
Enrolled Users
→ select
Enrollment Methods
To the right of
Guest Access
→
select
Gear Icon,
enter a password →
Save
.
Instructions for Guest Users
Follow the instructions above to allow guest access and optionally set a password.  Email a copy the URL of the course and the guest password to your visitor or write the password on the board in class and ask non-enrolled students to browse through the Moodle course catalog to find your course.

Go to the
Participants Tab
(under the course title)

Enrolled Users

Enrollment Methods

icon
on the

Is Guest Access Enabled?
You can tell whether Guest Access is enabled by checking for the following:
On your Moodle course page, go to the
Participants Tab
(under the course title)
In the pulldown menu labeled
Enrolled Users
→ select
Enrollment Methods
If
Guest Access
is in the list of Enrollment methods and it is not grayed out (the
Eye icon
does not have a slash through it), Guest Access is
enabled
.
If
Guest Access
is listed but the
Eye Icon
has a slash through it, Guest Access is
disabled
.
If
Guest Access
is not listed as an Enrollment method it is
disabled
Password setting
Follow the instructions above to check on guest access status.  Click the
Edit button (gear icon)
on the
Guest Access
row to see if a password is set.  If the
Password
field has a series of dots, a password is set.
