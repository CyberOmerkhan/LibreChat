Use a custom text style
On your computer, open a document in
Google Docs
.
Add some text and style it the way you want using the options found in the top toolbar.
Select that text.
Click
Format
.
Select
Paragraph styles
.
Choose the style you want to update e.g.
Normal text
or
Heading 1-6
.
Select
Update to match
.
All of the text in your document of that text type, such as "Heading 4" or "Normal text," will be updated to match your new style.
Set and change a default style
Set a default style
On your computer, open a document in
Google Docs
.
Set a default style: Click
Format
.
Select
Paragraph styles,
then select
Options
.
Finally, choose
Save as my default styles.
Apply a default style
On your computer, open a document in
Google Docs
.
Apply a default style: Click
Format
.
Select
Paragraph styles,
then select
Options
.
Finally, choose
Use my default styles
.
Restore a default style
On your computer, open a document in
Google Docs
.
Restore original to the Google style: Click
Format
.
Select
Paragraph styles,
then select
Options
.
Finally, choose
Reset styles
.
