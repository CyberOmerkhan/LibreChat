Click your
Start
button.

Search
box Type
\\swat-print

Find the
printer queue name from the table above.

Download and install the Mac Printer Drivers first.
Download Mac Printer Driver
Unzip and run the
SCF170_1326_AM.dmg
installer. This installer may warn that it cannot open because it is from an unidentified publisher.  You can launch the installer by control-clicking on the file and choosing "open".
Open
System Preferences
, select
Printing & Scanning
, and click the lock in the lower left corner to make changes.
Select the
+ (plus sign)
to add a printer
Click the
IP
tab, represented as
on newer versions of macOS.
In the Address field, type
swat-print
For Protocol, choose
Line Printer Daemon - LPD
In the
Queue
field, enter
prtsrv-500fieldhouse-sublimation.
Under
Name
, copy and paste name from
Queue
field.
Under
Use
, select the option for
Select Printer Software...,
and search
Epson SC-F100 Series.
Click
Add
. After you click Add, the Printer list window will reappear with the selected printer added to the list of printers.
